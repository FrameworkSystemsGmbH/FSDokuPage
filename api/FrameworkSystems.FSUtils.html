    GetMutex(String)

    Erzeugt einen Mutex mit dem übergebenen Namen und wartet ggf. darauf bis dieser von einem anderen Prozess wieder freigegeben wird. Beim Dispose des zurückgegebenen Objektes wird der Mutex freigegeben. Am besten mit einem using verwenden.

    using (FSUtils.GetMutex("MyMutex"))
{ /* aktion */ }
