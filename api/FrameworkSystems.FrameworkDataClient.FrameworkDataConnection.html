    Class FrameworkDataConnection

    Stellt eine Verbindung zu einer Datenbank dar. Abhängig vom Property ConnectionType kann eine Verbindung zu unterschiedlichsten DBMS-Systemen und über verschiedene Treiber hergestellt werden. Zur Zeit wird über eine Verbindung über System.Data.SqlClient (SqlServer) und System.Data.OleDb (Oracle u.ä.) unterschieden.
