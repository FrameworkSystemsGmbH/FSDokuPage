    Size

    Liest oder setzt die maximale Größe der Daten einer Spalte. Bei Unicode Anzahl der Zeichen und bei AnsiString Bytes. Wird außerdem für Binarys verwendet.

    Declaration
    public int Size { get; set; }
    Property Value
    Type Description
    System.Int32
    Remarks

    Für von der Länge her variable Datentypen bezieht sich Size auf die maximale Anzahl von Daten eines Strings die zum Server geschickt werden.

    Bei Nicht-String-Datentypes bzw. ANSI-STRINGS bezieht sich size auf die Byteanzahl.

    Bei Unicode-Strings bezieht sich Size auf die Anzahl der Zeichen.

    Falls Size nicht explizit gesetzt wird wird Size automatisch anhand des angegebenen Parameter-Wertes gesetzt.

    Für fixierte Datentypen (fixed width data types) wird Size ignoriert.

    For variable length data types, the Size property describes the maximum amount of data to transmit to the server. For example, Size can be used to limit the amount of data sent to the server for a string value to the first one hundred bytes.
