    AddProviderClone(FrameworkDataParameter)

    Erzeugt aus dem übergebenen Parameter einen Clone und fügt diesen in die Provider-Collection ein. Der Clone wird am übergebenen Parameter registriert. Dadruch werden Wert-Änderungen am Parameter auch in den Clonen angewendet.

    Dies ist nötig, wenn z.B. im Oracle-Provider derselbe Parameter mehrfach im Command vorkommt. Intern wird er dupliziert, aber nach außen gibt es nur einen FrameworkDataParameter, an dem auch nachträglich noch der Wert geändert und das Command erneut ausgeführt werden kann.
