    Class OracleExprConverterBase<T>

    Diese Klasse bildet die Oracle-Funktionen und Oracle-spezifische Syntax ab.

    Eigentlich gehört dieser Code in die Oracle-Provider. Er wäre aber in allen 3 Oracle-Providern identisch, dehalb hier als Basis-Klasse abgebildet.
