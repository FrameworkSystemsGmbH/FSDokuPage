    TryEnter(Object, String)

    Startet eine Rekursions-Absicherung für das übergebene Objekt und den übergebenen Key. Liefert null, wenn bereits eine Rekursion existiert.
    Immer mit einem using verwenden, um den Dispose() zu garantieren!

    Declaration

    Parameters
    Type Name Description
    System.Object instance

    Die Objekt-Instanz, die abgesichert werden soll
    System.String key

    Ein Key - für eine Objekt-Instanz können mehrere unterschiedliche Keys gleichzeitig abgesichert werden. Idealerweise wird als Key der der Name der aktuellen Methode bzw. des aktuellen Properties verwendet. In diesem Fall kann der Parameter weggelassen werden.
