    Class DataAdapters

    Klasse zur Definition von SqlDataAdaptern mit dem grafischen VS.NET-Editor. Sinnvoller Weise werden abgeleitete Klassen als Singleton-Klasse implementiert. Sie haben ein statisches Get-Property, welches ein Objekt der abgeleiteten Klasse enthält. z.B. 'public static readonly »DataAdapters« Get = new »DataAdapters«();'. Auf die SqlDataAdapter kann dann von jeder Klasse per »DataAdapters«.Get.»AdapterName« zugegriffen werden. Bei abgeleiteten Klassen ist folgendes zu beachten:

    • statisches Get-Property erstellen, welches ein Objekt der abgeleiteten Klasse enthält
    • this.InitializeComponent() muß im Konstruktor aufgerufen werden
    • SqlDataAdapter auf die zugegriffen werden soll müssen entsprechende Zugriffslevel beiten (internal oder public)

    JoinSqlDataAdapter(ref SqlDataAdapter, SqlDataAdapter)

    Verbindet einen Write-Adapter mit einem Read-Adapter. Hierdurch werden BaseObjects ermöglicht, die sich auf mehrere Tabellen beziehen. Hierbei wird ein Read-Adapter erzeugt welcher den kompletten select-Befehl beinhaltet. Ferner wird ein Write-Adapter erzeugt welcher das Speichern der Daten (insert, update und delete) in eine Tabelle ermöglicht. Diese Funktion fügt die Speichern-Befehle (insert, update und delete) des Write-Adapters dem Read-Adapter hinzu.

    JoinSqlDataAdapter(ref SqlDataAdapter, SqlDataAdapter[])

    Verbindet mehrere Write-Adapter mit einem Read-Adapter. Hierdurch werden BaseObjects ermöglicht, die sich auf mehrere Tabellen beziehen. Hierbei wird ein Read-Adapter erzeugt welcher den kompletten select-Befehl beinhaltet. Ferner werden Write-Adapter erzeugt welche das Speichern der Daten (insert, update und delete) in jeweils eine Tabelle ermöglichen. Diese Funktion verbindet die Speichern-Befehle (insert, update und delete) der Write-Adapter und fügt diese dem Read-Adapter hinzu. Die Write-Adapter sind in der Reihenfolge anzugeben, in der die Update- bzw. Insert-Befehle abzuarbeiten sind, die Delete-Befehle werden dann in der umgekehrten Reihenfolge abgearbeitet.
