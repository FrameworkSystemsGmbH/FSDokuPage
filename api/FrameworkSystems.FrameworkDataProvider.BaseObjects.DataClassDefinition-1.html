    Class DataClassDefinition<DataValuesType>

    GetSubClassDefinition(IDataValues)

    Diese Methode wird beim Instantiieren von DataRecords zur definierten Klasse aufgerufen. In Abhängigkeit von der übergebenen DataRow kann hier eine andere Sub-Klassendefinition für den zu instantiierenden DataRecord gesetzt werden. Der DataRecord wird als der in der Sub-Klassendefinition angegebe Typ instantiiert. Dies dient zur Abbildung einer Klassen-Generalisierung. Der DataRecord wird im DataStore unter der Vater(dieser)-Klassendefinition verwaltet, seine Feldinformationen erhält er jedoch von der abgeleiteten (hier ermittelten) Klassendefinition. Im Normalfall existiert keine Generalisierung und diese Methode gibt einfach diese Klassendefinition zurück. Diese Methode darf nie null zurück geben.
