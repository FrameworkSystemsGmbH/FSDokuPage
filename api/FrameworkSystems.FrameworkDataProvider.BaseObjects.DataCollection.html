    Class DataCollection

    Basis-Klasse einer Sammlung von Datenbank-Objekten. Ummantelt eine Table, auf deren Datensätze über die enthaltenen DataRecords zugegriffen werden kann. Beim Instantiieren wird ein SqlDataAdapter angegeben, mit dem per Fill- und Update-Method auf die Datenbank zugeriffen werden kann.

    DataCollection(DataRecord)

    Erstellt eine DataCollection mit genau einem Datensatz, der eigentlich zu einer anderen DataCollection gehört. Die so erstellte DataCollection ist lediglich eine Ummantelung des DataRecords und ist eng mit dessen Original-DataCollection verknüpft. Add und Remove-Befehle sollten nur an der Original-DataCollection durchgeführt werden.
