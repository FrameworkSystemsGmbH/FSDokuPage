    Class DataConnector

    Verwaltung der Datenbankanbindung der BaseObjects. In Klassenvariablen wird die Datenbankverbindung und eine Transaktionverwaltet. Zudem wird verwaltet ein DataConnector-Objekt den DataAdapter einer DataCollection.

    Transaction

    Transaktion welche für aktuelle Datenbankoperationen genutzt wird. Erzeugt wird eine Transaktion per DataConnector.BeginTransaction, beendet wird sie per Transaktion.Commit oder DataConnector.CommitTransaction, abgeprochen wird sie per Transaktion.Rollback oder DataConnector.RollbackTransaction. Bei Transaktion.Commit und Transaktion.Rollback muß anschließend noch die Datenbankverbindung geschlossen werden.

    Methods

    BeginTransaction()

    Startet eine Transaktion, welche mit Transaktion.Commit oder DataConnector.CommitTransaction beendet bzw. mit Transaktion.Rollback oder DataConnector.RollbackTransaction abgebrochen werden kann. Bei Transaktion.Commit und Transaktion.Rollback muß anschließend noch die Datenbankverbindung geschlossen werden. Falls bereits eine gestartete Transaktion existiert, wird keine neue gestartet, sondern weiterhin die bereits gestartete benutzt.
