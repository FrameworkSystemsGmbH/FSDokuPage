    Class DataFormsCursor

    Klasse zum Einstellen des Anwendungs-Cursors. DataFormsCursor merkt sich das Object, welches den Cursor gesetzt. Nur dieses kann den Cursor wieder freigeben bzw. einen anderen Cursor setzen
