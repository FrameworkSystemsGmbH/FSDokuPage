    IsDataSignatureValid(XmlNode, XmlDocument, Boolean)

    Checks the signature of the "Data" node, which has to be a sub node of parameter "node". If the signature is missing, false will be returned. The public key of this package will be used for verifying.

    You still have to check, wether the data node holds information about the packege, it was signed with!

    Declaration
    bool IsDataSignatureValid(XmlNode node, XmlDocument xmlDoc, bool showErrorMessages)
    Parameters
    Type Name Description
    System.Xml.XmlNode node

    The node from the xml document, which contains the "Data" node to be checked. For example the "License" node.
    System.Xml.XmlDocument xmlDoc

    The xml document.
    System.Boolean showErrorMessages

    On errors: If "showErrorMessages" is true, a message box with the error message will be displayed. If "showErrorMessages" is false, an exception will be thrown.

    SignDataNode(ref XmlNode, ref XmlDocument, Boolean)

    Signs the "Data" node (sub node of parameter "node") of the xml document using the private key of this package. This method may only be called for the active package! It uses the private key of the package.
