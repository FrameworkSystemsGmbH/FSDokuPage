    Class FSDisplayNameAttribute

    Dieses Attribute kann in Enum-Werten verwendet werden, um einen DisplayName anzugeben. Dieser kann mit der statischen Methode GetFromEnum(Enum) ermittelt werden.

    GetFromEnum(Enum)

    Liefert den DisplayName eines Enum-Members. Wenn das Attribut nicht definiert ist, dann wird der Name des Enums zurückgegeben.
    GetOrderFromEnum(Enum)

    Liefert die Order eines Enum-Members. Wann das Attribut oder die Order nicht angegeben sind, dann wird 0 zurückgegeben.
