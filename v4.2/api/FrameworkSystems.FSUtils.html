    BuildExceptionString(Exception)

    Baut die Details einer Exception in einem String zusammen. Ist ein Workaround, weil in dornet 4.0 das Verhalten von System.Exception.ToString() geändert wurde.
    EscapeCommandLineArgs(IEnumerable<String>)

    Wandelt die ünergebenen String-Argumente in einen string mit Command-Line-Arguments um. Dabei werden Leerzeichen, \ und " escaped.
    EscapeCommandLineArgs(String)

    GetMutex(String)

    Erzeugt einen Mutex mit dem übergebenen Namen und wartet ggf. darauf bis dieser von einem anderen Prozess wieder freigegeben wird. Beim Dispose des zurückgegebenen Objektes wird der Mutex freigegeben. Am besten mit einem using verwenden.

    using (FSUtils.GetMutex("MyMutex"))
{ /* aktion */ }
    PerfCounterStart()
