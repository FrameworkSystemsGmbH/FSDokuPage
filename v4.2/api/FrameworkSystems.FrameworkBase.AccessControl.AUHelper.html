    ChangePassword(String, String)

    Ändert das Passwort des momentan angemeldeten Benutzers.

    ChangePasswordChecked(String, String, String, out AUHelper.Message, String)

    Ändert das Passwort für den angegebenen Benutzer unter Angabe einer Login Information (beliebiger Text, der vom Authentication Service geloggt wird, darf null sein). In der als out Parameter angegebenen Message wird der Erfolg bzw. Grund für den Misserfolg zurückgegeben.
