    FormatMLText(MLKey, Object[])

    Übersetz einen MLKey in einen Text. Dabei wird ein String.Format(String, Object[]) mit den übergebenen Argumenten ausgeführt. Für die Fomatierung der übergebenen Argumente wird die Client-Culture verwendet.

    Declaration
    string FormatMLText(MLKey mlkey, params object[] args)
    Parameters
    mlkey MLKey

    MLKey aus dem Wörterbuch

    args Object[]
    Returns
    String

    Formatierter Text

    Examples
    FSstring sFormattedText = this.Global.FormatMLText( MLKeys.LBL_MitFormat, "Text1", lngAmount );
