    4.3 Beta 2 (vorr. 30.10.2020)

    Neuerungen

    • Umstellung auf PropertyChanged-Event

    Korrigierte Fehler

    • 2020094047 - Code-Generierung: Für Individual Properties wird autoamtisch ein Setter generiert, der bei jeder Wertzuweisung den Getter des Properties aufruft, um das Ergebnis gegen den neuen Wert zu prüfen. Das kann ggf. die Performance beeinträchtigen.

      Note

      Dieser Aufruf wurde entfernt. Jetzt läuft jeder Aufruf des Setters direkt in den Entwickler-Code.

    Korrigierte Fehler

    In der Version 4.3 sind alle Fehler-Korrekturen bis einschließlich Framework Studio 4.2.10 enthalten. Zusätzlich wurden die folgenden Bugs behoben.

    • 2020052729 - Java-Client: Im Dialog für die Eingabe der Spalten-Filter sollte der Fokus direkt in das Eingabe-Feld gesetzt werden. Nach Eingabe eines Wertes soll der Filter direkt angewendet werden.

    • 2017089506 - IDE: Wenn man in einem Codeeditor die Suche eingeschränkt auf den selektierten Bereich ausführt, wird immer Case-sensitiv gesucht, auch wenn die Checkbox nicht gesetzt ist. Find and Replace ist ebenfalls betroffen.

    • 2020042323 - IDE: Beim Erstellen einer individual Datenbank-Connection wird ein neuer (leerer) Eintrag erstellt. Es sollte der vorhandene Eintrag kopiert und als Individual (editable) angelegt werden.

    • 2020073102 - IDE: Wird in einer DBTable die letzte Column gelöscht, dann wird eine NullReferenceException angezeigt.
