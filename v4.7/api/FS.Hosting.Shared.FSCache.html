    Session(IFSCache)

    Liefert den MemoryCache für die aktuelle Session. Die Session wird mithilfe des Token vom GlobalContext identifiziert.

    Der Cache wird verworfen, wenn

    • Am Ende einer Session SessionCancellationToken abgebrochen wird oder
    • 1 Stunde nicht auf den Cache zugegriffen wird.

    Static(IFSCache)

    Liefert den MemoryCache für den Static-Scope.
